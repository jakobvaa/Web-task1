Flexbox og CSS-Grid

Flexbox og CSS-Grid er gode metode for å strukturere plasseringen til HTML elementer på en nettside. Denne layout modellen blir sett på som en erstatning for positioning og floats. Ved hjelp av "layout modellene" kan man enkelt bestemme hvor elementene skal plasseres og avstanden mellom de. Flexbox er en-dimensjonal, det vil si at alle elementer går i samme retning. Retningen er "main axis" til flex-containeren. Elementene som ligger langs denne aksen kan vokse for å ta opp ubrukt plass eller krympe, og mye mer! Disse elementene som ligger i flex-containeren blir kalt flex-items.
CSS-Grid er en layout modell som kan brukes for å strukturere HTML elementene i to-dimensjoner. Dette gjør det enkelt å lage komplekse strukturer på nettsiden din.

På denne nettsiden valgte eg å bruke CSS-Grid. Dette gjorde eg fordi det gir meg muligheten til å enkelt lage komplekse strukturer av elementene på nettsiden. Kunne nok brukt flexbox også på denne nettsiden, men CSS Grid sine muligheter gjorde det fristande å lære. Brukte også Flexbox i det blå/ kvit stripete div'en over for å teste det ut.



HTML-5 Canvas og SVG

Dette er to teknologier som kan brukes for å tegne grafikk på nettsider. Sjølv om dei har like mål har dei to ulike framgangsmåter for å gjøre dette. SVG er basert på XML og alle delene av kunstverkene vil være objekter. I HTML5 Canvas derimot er det bare en container der alle pixlene blir tegnet på nettsiden. Dersom man skal endre på kvert enkelt objekt i canvas er dette vanskeligere, ettersom man må kontrollere state feks. lagre objektene. I SVG er objektene DOM objekter, så man kan bruke feks. click handlers.

Eg valgte å bruke HTML5 canvas ettersom eg ønsket å manipulere mange objekter. Det er ikkje optimalt å ha mange objekter i SVG ettersom alle elementene er elementer i DOMen.



jQuery

Er ett bibilotek til Javascript som rett og slett gjør det enklere å bruke Javascript. Biblioteket gir deg metoder som ville trengt mange linjer kode å skrive i Javscript. Med jQuery kan man blant annet gjøre Html/ Dom manipulering, css manipulering, effekta og animasjona med meir. Enda en fordel med jQuery er at det fugerer med alle de store browserene. På denne nettsiden har eg brukt jQuery til å legge til å forandre css atributtene til denne diven fra å være skjult til å bli vist.



Cross-browser-testing

Denne typen testing er viktig ettersom det ikkje er en standard på hvordan nettlesere implementeres. Ulike nettlesere kan vise samme nettside på ulike måter og støtte ulyke typer teknologier. Eg har testa min nettleser i mozilla firefox og Chromium og sikre meg at nettsiden visest på samme måte.
